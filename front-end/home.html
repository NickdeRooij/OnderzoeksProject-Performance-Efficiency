Filtering Performance Experiment

This project investigates the performance differences between client-side filtering and server-side filtering in web applications.

Large datasets can impact performance depending on where filtering logic is executed. This dashboard allows you to experiment with different dataset sizes and compare the performance of both approaches.

Open Dashboard
Experiment overview
  • Datasets from 1.000 to 1.000.000 records
  • Client filtering using JavaScript
  • Server filtering using .NET API
  • Performance measured in milliseconds

Client-side filtering

Filtering is performed in the browser using JavaScript after the dataset has been loaded.

Server-side filtering

Filtering is handled by the backend API before the data is returned to the client.

Performance comparison

The dashboard measures execution time to determine which approach performs better for different dataset sizes.
